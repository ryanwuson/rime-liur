# 蝦米輸入法 Rime 方案

基於 Rime 輸入法引擎的蝦米輸入法方案，提供多元之輸入體驗。

## 快速開始

請參閱 [完整使用說明](user-guide.md) 了解所有功能。

## 主要功能

| 功能 | 快捷鍵/引導鍵 | 說明 |
|:-----|:-------------|:-----|
| 英文輸入 | `Ctrl + /` | 臨時英文輸入 |
| 造詞模式 | `;` | 臨時造詞功能 |
| 同音選字 | `'` | 選中候選字後按 `'`，顯示同音字 |
| 讀音查詢 | `;;` | 輸入字碼查讀音 |
| 注音輸入 | `';` | 注音輸入中文 |
| 符號清單 | `` ` `` | 50+ 分類符號 |
| 快打模式 | `,,sp` | 提示可用簡碼 |
| 萬用查字 | `,,wc` + `?` | 模糊查詢字碼 |
| 查碼模式 | `Ctrl + '` | 顯示字碼，學習拆字 |
| 按鍵說明 | `,,h` | 顯示所有快捷鍵說明 |

## 安裝

1. 下載本專案所有檔案
2. 將檔案複製到 Rime 使用者資料夾
3. 重新部署 Rime
